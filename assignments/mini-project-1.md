# Mini Project 1 — Responsive Webpage

Due: End of Week 05
Weight: 10%

## Brief

Build a small multi-section webpage that adapts to mobile and desktop using modern CSS layout techniques (Flexbox/Grid) and media queries. Content topic is your choice (e.g., a simple product page, event page, or personal landing page).

## Requirements

-   Semantic HTML structure with landmarks
-   Mobile-first CSS with at least one breakpoint
-   Use Flexbox or Grid for layout (or both)
-   Accessible navigation with clear focus states
-   Images with appropriate alt text and responsive behavior
-   Deployed via GitHub Pages or similar

## Deliverables

-   GitHub repo link with source code
-   Live demo URL
-   README with:
    -   Project description and screenshots
    -   How to run locally
    -   Attributions/credits

## Rubric (10 points)

-   Structure & Semantics (2): Landmarks and heading outline are logical
-   Responsive Layout (3): Works well on mobile and desktop; uses media queries
-   Accessibility Basics (2): Alt text, focus states, color contrast considered
-   CSS Quality (2): Organized, avoids repetition, uses modern features (Flex/Grid)
-   Polish (1): Deployed, README complete, no obvious broken links/images
