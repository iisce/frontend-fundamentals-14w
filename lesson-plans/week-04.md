# Week 04 — CSS Layout I (Index)

This week focuses on foundational CSS layout techniques, including box model, positioning, and multi-column layouts.

## Overview

-   Focus: CSS box model, positioning, and layout strategies
-   Outcomes:
    -   Understand and apply the CSS box model (margin, border, padding, content)
    -   Use positioning properties (`static`, `relative`, `absolute`, `fixed`, `sticky`)
    -   Create multi-column and layered layouts using floats and positioning

## Sessions

### Session 10 — Box Model and Spacing

-   Summary: Anatomy of the box model, controlling spacing with margin and padding, and visualizing layout boundaries.
-   Link: [week-04/session-10.md](week-04/session-10.md)

### Session 11 — Positioning and Layering

-   Summary: Positioning schemes (`static`, `relative`, `absolute`, `fixed`, `sticky`), z-index, and stacking contexts for layered designs.
-   Link: [week-04/session-11.md](week-04/session-11.md)

### Session 12 — Multi-Column Layouts

-   Summary: Creating multi-column layouts with floats, clearing techniques, and modern alternatives.
-   Link: [week-04/session-12.md](week-04/session-12.md)

## Deliverables This Week

-   Complete box model and positioning labs
-   Build a multi-column layout using floats or modern CSS
-   Optional: Document your layout’s acceptance criteria in your project README
