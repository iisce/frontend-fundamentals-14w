# Week 10 — DOM & Events + Mini Project 3 (Index)

This week covers DOM manipulation, event handling, and culminates in building an interactive application.

## Overview

-   Focus: Document Object Model (DOM), event handling, and interactive web applications
-   Outcomes:
    -   Understand the DOM tree structure
    -   Select and manipulate DOM elements
    -   Handle user events effectively
    -   Build interactive web applications
