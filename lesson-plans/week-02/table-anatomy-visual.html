The Code Behind It

<table>
  <caption>Monthly Sales Report ‚Äî Q4 2025</caption>

  <thead>
    <tr>
      <th scope="col">Month</th>
      <th scope="col">Revenue</th>
      <th scope="col">Expenses</th>
      <th scope="col">Profit</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <th scope="row">October</th>
      <td>$45,000</td>
      <td>$32,000</td>
      <td>$13,000</td>
    </tr>
    
  </tbody>

  <tfoot>
    <tr>
      <th scope="row">Total</th>
      <td>$165,000</td>
      <td>$108,000</td>
      <td>$57,000</td>
    </tr>
  </tfoot>
</table>

Key Takeaways

  • <caption> comes first, right after <table>
  • <thead> contains column headers with scope="col"
  • <tbody> contains data rows; first cell often uses scope="row"
  • <tfoot> is optional, used for totals or summaries
  • Each <tr> (table row) must have the same number of cells
  • Use <th> for headers, <td> for data
  • The scope attribute tells screen readers what each header applies to
