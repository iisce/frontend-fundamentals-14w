# Week 01 — HTML Foundations I (Index)

Welcome to Week 01. This week you’ll set up your environment, learn HTML fundamentals, and practice authoring semantic, accessible content.

## Overview

-   Focus: Core HTML, content structure, semantics, and accessibility basics.
-   Outcomes:
    -   Author valid HTML5 documents with semantic elements
    -   Use headings, lists, links, images (with alt), and basic page structure
    -   Understand landmarks and why semantics matter for accessibility

## Sessions

### Session 01 — Introduction + HTML Basics

-   Summary: How the web works, minimal HTML5 document, and content tags (headings, paragraphs, links).
-   Key outcomes:
    -   Create a valid HTML5 page with correct head/body structure
    -   Use headings and paragraphs for meaningful content
    -   Navigate DevTools at a high level
-   Materials: [../templates/project-scaffold/](../templates/project-scaffold/)
-   Link: [week-01/session-01.md](week-01/session-01.md)

### Session 02 — Text and Media

-   Summary: Semantic text elements, lists, links, paths, and images with accessible alt text.
-   Key outcomes:
    -   Create lists (ul/ol) and apply emphasis correctly (em/strong)
    -   Add absolute/relative/hash links and verify paths
    -   Include images with meaningful alt text
-   Link: [week-01/session-02.md](week-01/session-02.md)

### Session 03 — Structure and Semantics

-   Summary: Page structure with landmarks (header, nav, main, section, article, aside, footer) and accessibility basics.
-   Key outcomes:
    -   Organize content with semantic layout elements
    -   Produce a logical heading outline with one h1
    -   Understand landmarks and the accessibility tree
-   Link: [week-01/session-03.md](week-01/session-03.md)

## Deliverables This Week

-   Complete Session 01–03 labs inside the scaffold
-   Homework: Article page (S01), Text-to-HTML conversion (S02), Semantic structure (S03)
-   Optional: Publish your page with Live Server or GitHub Pages for feedback
