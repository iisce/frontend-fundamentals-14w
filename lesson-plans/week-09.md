# Week 09 — JavaScript Fundamentals (Index)

This week introduces JavaScript programming fundamentals, covering variables, data types, operators, and control flow.

## Overview

-   Focus: JavaScript basics, variables, data types, operators, and control structures
-   Outcomes:
    -   Understand JavaScript's role in web development
    -   Declare and use variables with `let`, `const`, and `var`
    -   Work with primitive and reference data types
    -   Use operators and control flow statements
