# Week 06 — Responsive Design & Flexbox/Grid (Index)

This week focuses on responsive design principles, CSS Grid layout system, and combining modern layout techniques.

## Overview

-   Focus: Responsive design, media queries, CSS Grid, and combining Flexbox with Grid
-   Outcomes:
    -   Understand responsive design principles and mobile-first approach
    -   Use media queries effectively for different screen sizes
    -   Master CSS Grid for two-dimensional layouts
    -   Combine Flexbox and Grid for complex layouts

## Sessions

### Session 16 — Responsive Design & Media Queries

-   Summary: Responsive design principles, viewport meta tag, media queries, breakpoints, and mobile-first vs desktop-first approaches.
-   Link: [week-06/session-16.md](week-06/session-16.md)

### Session 17 — CSS Grid Fundamentals

-   Summary: Grid container and item properties, defining rows and columns, grid template areas, and explicit vs implicit grids.
-   Link: [week-06/session-17.md](week-06/session-17.md)

### Session 18 — Combining Flexbox & Grid

-   Summary: When to use Flexbox vs Grid, combining both for complex layouts, building complete page layouts, and real-world patterns.
-   Link: [week-06/session-18.md](week-06/session-18.md)
