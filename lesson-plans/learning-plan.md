## JavaScript Learning Roadmap (Weeks 07–10)

This is the comprehensive 4-week JavaScript journey. Each week builds on the previous one.

### Week 07 — JavaScript Fundamentals I: Syntax, Variables & Data Types

> _"Your first lines of JavaScript — understanding the language from the ground up."_

| Session | Topic | Duration | Highlights |
|---------|-------|----------|------------|
| 19 | Introduction to JavaScript & Dev Tools | 90 min | What is JS, where it runs, adding JS to HTML, browser console, console.log(), alert(), prompt() |
| 20 | Variables, Constants & Data Types | 90 min | let, const, var, primitive types (string, number, boolean, null, undefined, symbol, BigInt), typeof, type coercion |
| 21 | Operators, Expressions & Control Flow | 90 min | Arithmetic/comparison/logical/assignment operators, truthy/falsy, if/else if/else, switch, ternary, template literals |

**Deliverables:** JavaScript exercises, simple interactive greeting program, temperature converter

---

### Week 08 — JavaScript Fundamentals II: Functions, Arrays & Objects

> _"Writing reusable code and working with collections of data."_

| Session | Topic | Duration | Highlights |
|---------|-------|----------|------------|
| 22 | Functions & Scope | 90 min | Function declarations, expressions, arrow functions, parameters, return values, default params, scope (global/local/block), hoisting |
| 23 | Arrays & Iteration | 90 min | Creating arrays, index access, .push(), .pop(), .splice(), .slice(), for, for...of, .forEach(), .map(), .filter(), .find(), .reduce() |
| 24 | Objects & JSON | 90 min | Object literals, properties, methods, dot vs bracket notation, destructuring, spread/rest, Object.keys(), Object.values(), JSON stringify/parse |

**Deliverables:** Function challenge set, array manipulation exercises, student grade tracker app, JSON data processor

---

### Week 09 — The DOM & Event-Driven Programming

> _"Making web pages alive — selecting, changing, and responding to user actions."_

| Session | Topic | Duration | Highlights |
|---------|-------|----------|------------|
| 25 | DOM Fundamentals & Selection | 90 min | What is the DOM, DOM tree, getElementById, querySelector, querySelectorAll, traversal (parentElement, children, nextElementSibling) |
| 26 | DOM Manipulation & Styling | 90 min | textContent, innerHTML, setAttribute, classList (add/remove/toggle), creating/appending/removing elements, style property, dynamic lists |
| 27 | Events, Forms & User Interaction | 90 min | addEventListener, event object, event.preventDefault(), event bubbling & delegation, keyboard/mouse/form events, input validation in JS |

**Deliverables:** Interactive to-do list, dynamic color palette generator, form validator with real-time feedback

---

### Week 10 — ES6+, Async JavaScript & Mini Project 2

> _"Modern JavaScript and fetching data from the web."_

| Session | Topic | Duration | Highlights |
|---------|-------|----------|------------|
| 28 | Modern ES6+ Features | 90 min | Arrow functions deep dive, destructuring, spread/rest, modules (import/export), let/const best practices, shorthand properties |
| 29 | Asynchronous JavaScript & Fetch API | 90 min | Sync vs async, callbacks, Promises (.then(), .catch(), .finally()), async/await, try/catch, Fetch API, working with public APIs |
| 30 | Mini Project 2: Interactive App | 90 min | Build a complete interactive app combining DOM, events, ES6+, and Fetch (e.g., weather app, movie search, quiz game) |

**Deliverables:** Refactored code using ES6+, API data fetcher, **Mini Project 2: Interactive JavaScript App (due end of Week 10)**
